ESP8266 DHT11 → Google Sheets

NodeMCU firmware reads a DHT11 and logs temperature and humidity to Google Sheets via PushingBox.

Wiring

  1. DHT11 VCC → NodeMCU 5V (Vin)
  2. DHT11 GND → GND
  3. DHT11 data → D4 (GPIO2)

Flow: DHT11 → ESP8266 → PushingBox (HTTP) → Google Apps Script (HTTPS) → Google Sheet

Test your PushingBox device

Enter your PushingBox device ID and sample sensor values. No secrets are stored — snippets are generated in your browser.

Fill in your device ID and click Generate.

Setup checklist

  1. Create a Google Sheet + deploy Apps Script web app
  2. Add a PushingBox CustomURL service pointing at your script URL
  3. Copy the PushingBox device ID into firmware secrets.h
