Sheets logger for your climate node

NodeMCU firmware samples a DHT11 on a fixed interval and pushes temperature and humidity rows to Google Sheets through PushingBox — no cloud broker required.

⎔ Wiring

  1. DHT11 VCC → NodeMCU 5V (Vin)
  2. DHT11 GND → GND
  3. DHT11 data → D4 (GPIO2)
DHT11 → ESP8266 → PushingBox → Apps Script → Sheet

▣ Test console

Enter your PushingBox device ID and sample sensor values. Nothing is stored server-side — requests are built in your browser. 

Fill in your device ID and click Generate.

◎ Setup checklist

  1. Create a Google Sheet and deploy the Apps Script web app
  2. Add a PushingBox CustomURL service pointing at your script URL
  3. Copy the PushingBox device ID into firmware secrets.h
